    • O GitHub é uma rede social de desenvolvedores. A primeira parte do nome, “Git” é por causa da utilização do sistema de controle de versão e a segunda parte, “Hub” tem a ver com a conexão entre profissionais de programação de qualquer lugar do mundo.

    comando do Git

    o sistema de controle de versão moderno mais usado no mundo hoje é o Git. O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux.

    Visualiza os arquivos que foram alterados

    git status

    prepara um arquivo para ser enviado para o github

    git add nome_de_um_arquivo

    Gera a versão das modificacao

    git commit -m "uma mensagem legal do que foi feito nos arquivosnesta versao"

    Enviado para o github

    git push origin main
